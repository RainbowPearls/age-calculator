# age-calculator

A simple and modern **Age Calculator** built using HTML, CSS, and JavaScript.

It calculates your current age and shows how many days are left until your next birthday.



## Features

-  Accurate age calculation
-  Next birthday countdown
-  Instant result display
-  Mobile responsive design
-  Clean and modern UI
-  Invalid date prevention



## Technologies Used

- HTML5
- CSS3
- JavaScript
